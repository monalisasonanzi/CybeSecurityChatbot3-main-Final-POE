# CyberSecurityChatbot2

## Overview

CyberSecurityChatbot2 is a WPF (.NET) Cybersecurity Awareness Chatbot developed in C#. The chatbot helps users learn about online safety and cybersecurity threats.

It uses programming concepts such as Object-Oriented Programming (OOP), dictionaries, lists, memory recall, sentiment detection, and a graphical user interface (GUI).

## Features

* Answers cybersecurity-related questions
* Memory recall (remembers user name)
* Sentiment detection
* Modern chatbot GUI
* Response variation using lists
* Default responses for unknown questions
* Logo support
* Text-to-Speech support

## Topics Covered

The chatbot can answer questions about:

* Phishing
* Malware
* Password Safety
* Privacy
* Safe Browsing
* Scams
* Hackers
* Cyberbullying
* 2FA (Two-Factor Authentication)

## Technologies Used

* C#
* WPF (.NET)
* XAML
* Visual Studio 2026
* OOP Principles

## How to Run

1. Open the `.sln` file in Visual Studio.
2. Build the solution.
3. Press `Ctrl + F5` to run the chatbot.
4. Ask cybersecurity questions.

Example:

* What is phishing?
* Tell me about malware
* Password safety tips
